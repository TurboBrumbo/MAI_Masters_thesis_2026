# 🚁 Интеллектуальная аналитическая система тактико-технических характеристик гражданских БПЛА

---

## Описание

Программный комплекс предназначен для автоматизированного анализа, сегментации, прогнозирования реальных полётных характеристик и многокритериального ранжирования гражданских беспилотных летательных аппаратов.

Система объединяет:
- генерацию синтетических данных на основе физических моделей;
- предобработку и нормализацию признаков;
- иерархическую плотностную кластеризацию (HDBSCAN) для выделения рыночных ниш;
- обучение ансамблевых моделей машинного обучения (Random Forest, XGBoost) для прогноза реального времени полёта;
- объяснение предсказаний с помощью SHAP (Explainable AI);
- многокритериальное принятие решений (AHP + TOPSIS / VIKOR) для выбора оптимальной платформы с учётом экспертных предпочтений.

Реализован интерактивный веб-интерфейс на базе Streamlit, позволяющий проводить полный цикл анализа без написания кода.

---

## Стек технологий

- Python 3.12.10
- Streamlit, Plotly, Matplotlib – визуализация и интерфейс
- Pandas, NumPy, Scikit-learn – обработка данных и машинное обучение
- XGBoost, SHAP – продвинутое моделирование и интерпретация
- HDBSCAN – кластеризация
- PyMCDM – многокритериальный анализ

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <url>
   cd <repo>

2. Создайте и активируйте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate      # Linux/Mac
    venv\Scripts\activate         # Windows

3. Установите зависимости:
    ```bash
    pip install -r requirements.txt

4. Запустите приложение:
    ```bash
    streamlit run app.py

5. Откройте браузер по адресу http://localhost:8501, если он не открылся автоматически
